# ManSio

[🇺🇸 English Version](README.md)

Инженер-практик. Техник-строитель (ПГС) по образованию, системный исследователь по призванию.

Я строю программные системы с тем же уровнем ответственности, с которым строят здания: с запасом прочности, логикой и четкой документацией. Мой подход — **learning-by-doing**, подкрепленный жесткими инженерными протоколами и "живой" документацией.

### 🏗 Мои проекты
* **[MSCodeBase](https://github.com/ManSio/mscodebase-intelligence)** — развитие расширений для Zed IDE. Мой основной «полигон», где я внедряю AST-aware поиск, LanceDB и сложные пайплайны индексации.
* **[gemma_agent](https://github.com/ManSio/gemma_agent)** — эксперименты с AI-агентами. Исследование границ того, что может делать LLM без надзора.

### ⚙️ Инженерный манифест
Я не просто пишу код — я создаю рабочие среды, подчиняющиеся принципам:
- **Clean Architecture** (принцип «Тумблера»: технология не должна диктовать архитектуру).
- **Transparency** (каждое архитектурное решение обосновано экспериментом, зафиксированным в `EXPERIMENTS_LOG.md`).
- **Safety** (код без тестов — это недострой).
- **Living Documentation** (код и документация синхронизированы в одном коммите).
